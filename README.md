# fulltimeCalendar

Track and visualize your full-time commitment schedule.

## Overview

This tool helps you manage and visualize your full-time work schedule with customizable calendars and analytics.

## Features

- Weekly schedule visualization
- Time tracking integration
- Export to various calendar formats
- Progress analytics

## Installation

```bash
npm install fulltime-calendar
```

## Usage

```javascript
import { createCalendar } from 'fulltime-calendar';

const calendar = createCalendar(config);
calendar.render('#schedule');
```
